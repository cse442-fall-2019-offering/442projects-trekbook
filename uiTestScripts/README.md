Populate the file with the following table template

|-------|--------------|-----------------|
| 1     | Do this test | Get this result |
| 2     |              |                 |
| 3     |              |                 |

Use https://www.tablesgenerator.com/markdown_tables# if you would like to first and copy it into the md
